# Driver's License Tracking – Mobile User Guide

**Website:** [drivers.siam-ez.com](https://drivers.siam-ez.com)

Short guide for checking driver's license application status on a phone or tablet.

---

## Open the site

1. In your mobile browser, go to: **drivers.siam-ez.com**
2. Bookmark the page for quick access.

---

## Check application status

1. Tap **ติดตามสถานะใบขับขี่ / Track Status** on the home page.
2. Enter either:
   - **Passport number** (e.g. PA1234567), or  
   - **Application ID** (e.g. TH-DL-2026-0001)
3. Tap **ตรวจสอบ / Search**.

---

## Understanding the result

- **If a record is found:** You’ll see the applicant’s name, passport, application ID, and a **status timeline** (e.g. Application Received → Document Verification → Processing → License Issued → Completed). Any **Remark / หมายเหตุ** is shown at the bottom.
- **If no record is found:** You’ll see a message asking you to check the number and try again. Re-enter the passport or application ID and search again.

---

## Tips for mobile

- Use **portrait** or **landscape**; the page works in both.
- Make sure you type the passport or application ID correctly (no extra spaces).
- You can search again anytime with a new number.
